# Project Aegis — superpowers *documentation*

This directory holds **project-specific** specs and implementation plans produced with superpowers-style workflows (brainstorm → spec → plan → execute).

It is **not** the [obra/superpowers](https://github.com/obra/superpowers) plugin. For the global methodology and skill library, see:

- `docs/engineering/superpowers-setup.md`
- `.\tools\install-superpowers.ps1`

## Contents

| Path | Purpose |
|------|---------|
| `specs/` | Locked design decisions (delegation, DATA P0, sim modes, …) |
| `plans/` | Checkbox implementation plans referencing specs |
| `brainstorms/` | Early ideation notes |

When a plan says “REQUIRED SUB-SKILL: superpowers:subagent-driven-development”, load that skill from your global install (`~/.cursor/skills/subagent-driven-development/SKILL.md`).
