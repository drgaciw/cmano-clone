1. Executive verdict

Headless .NET is Linux-first and largely OS-agnostic. Buildkite hosted Linux agents own the blocking merge gate. Windows local verification is a thinner PowerShell mirror.

Unity players are dual-target in API but CI-gated on neither OS. Linux is the documented batchmode example and the only Editor menu shortcut; Windows is assumed for interactive Editor automation via hardcoded Unity.exe paths.

Core sim C# has no OS platform branching — gaps are tooling, Editor host, and player packaging.
